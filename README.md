## Improvements:

Over the original [Emoji Contact](https://github.com/Hysterelius/EmojiCP), there are a number of improvements:

- Mobile friendly 📱
- Much faster 🔥
- Looks better 😎
- Easier to interact with 🤳
- Latest emoji 🤩

Uses [Twitter Emojis](https://github.com/twitter/twemoji).
